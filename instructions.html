Инструкция по подключению

1. После оплаты

После подтверждения платежа на указанный email придёт письмо с логином и паролем для приложения HavenLink, а также ссылка на страницу загрузки.

2. Установка приложения

  • Android: скачайте APK с сайта havenlink.ru/download.html, разрешите установку из неизвестных источников и установите приложение.
  • Windows: скачайте архив с сайта, распакуйте и запустите HavenLink.exe. При первом подключении Windows может запросить права администратора для установки VPN-драйвера.

3. Вход в приложение

  1. Откройте приложение HavenLink.
  2. Введите логин и пароль из письма.
  3. Выберите сервер из списка (рекомендуется ближайший к вам по географии).
  4. Нажмите кнопку Подключиться.

4. Android: разрешение VPN

При первом подключении Android покажет системный запрос на создание VPN-подключения. Нажмите Разрешить / OK. Без этого разрешения туннель не заработает.

5. Проверка подключения

После подключения в приложении появится статус «Подключено». Можно открыть браузер и проверить свой IP на любом сервисе — он должен совпадать с выбранной страной сервера.

6. Отключение

Для отключения VPN нажмите кнопку Отключиться в приложении или отключите VPN в системных настройках устройства.

7. Поддержка

По любым вопросам пишите на support@havenlink.ru. В письме укажите логин и описание проблемы.
