# HavenLink Landing

Статический одностраничный сайт + страница оплаты для VPN-сервиса HavenLink.

## Файлы

- `index.html` — лендинг (RU/EN)
- `checkout.html` — страница оформления подписки
- `success.html` / `cancel.html` — страницы после оплаты
- `styles.css` — стили
- `assets/logo.png` — логотип
- `oferta.html`, `privacy.html`, `terms.html` — юридические страницы

## Домен

Настройте в настройках репозитория GitHub Pages custom domain: `havenlink.ru`.

> В `checkout.html` API backend указывается автоматически: `http://localhost:8000` на локалхосте, `https://api.havenlink.ru` в продакшене.
